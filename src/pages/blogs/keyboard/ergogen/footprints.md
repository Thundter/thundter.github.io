## nice_nano reversable footprint

https://github.com/ceoloide/ergogen-footprints

This library includes:

Reversible Nice!Nano v2 footprint with correct pin alignment. 
Support for KiCad 8.
Fixes for silkscreen inversion and DRC (Design Rule Check) errors.
Optional traces and vias for routing convenience. 
3D model support (added via community contributions).
